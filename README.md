# PySudokuSolver

Python script to solve a Sudoku.

## Project setup

Get into the project folder:
```bash
cd PySudokuSolver
```

Create a new environment:

```bash
virtualenv --python="/usr/local/bin/python3" env
```

Get into that environment:

```bash
source env/bin/activate 
```

## Usage

Run:

```bash
python3 app/src/main.py
```

## Versions

* Python 3.8.10
* Ubuntu 20.04.5
